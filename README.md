# GVA PharmaTrace Hub - Web FISABIO

Sitio web estatico del proyecto GVA PharmaTrace Hub para FISABIO.

## Estructura

- `Inicio.dc.html`: pagina principal.
- `Espacio-de-Datos.dc.html`: pagina de espacio de datos.
- `Etica-y-Seguridad.dc.html`: pagina de etica y seguridad.
- `Equipo.dc.html`: pagina de equipo.
- `Contacto.dc.html`: pagina de contacto.
- `assets/`: imagenes, logotipos y recursos visuales.
- `_ds/`: sistema de diseno y estilos compartidos.
- `uploads/`: documentos y materiales de contenido.

## Visualizacion local

Abrir `index.html` en el navegador. Este archivo redirige automaticamente a `Inicio.dc.html`.

## Despliegue temporal

Este proyecto se puede publicar como sitio estatico con GitHub Pages, Netlify o Vercel.
